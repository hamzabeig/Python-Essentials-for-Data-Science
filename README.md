## 📌 Course Overview

This course is designed to introduce students to Python libraries used for:
- Data manipulation
- Numerical computing
- Scientific computing
- Data visualization  

By the end, students will be able to work with real datasets and build basic data analysis workflows.

## 📦 Expected Outcomes

After completing this course, students will:
- Use NumPy for numerical operations  
- Manipulate data using Pandas  
- Build visualizations using Matplotlib  
- Understand basic SciPy concepts  
- Complete a mini project  

---

## 🚀 Final Project Idea

Analyze a dataset (e.g., sales or student data):

1. Load dataset  
2. Clean missing values  
3. Perform analysis  
4. 📊 Create visualizations (charts, graphs)
